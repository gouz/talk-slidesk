Commencé comme projet perso sans prétention, SliDesk commence petit à petit avoir des étoiles (GitHub) dans les yeux.

Des outils pour faire des présentations, il en existe plein, alors pourquoi en faire un autre ?

Je vous propose un REX sur la création d'un projet opensource, sur les craintes, les ascenseurs émotionnels, les travers dans lesquels il ne faut pas tomber.

Maintenant le projet évolue et dispose d'un "hub" pour la communauté et favoriser son émancipation.

Je vous propose également de faire une démonstration de l'outil pour essayer de vous convaincre à basculer dans le monde du SaC (Slides as Code).
